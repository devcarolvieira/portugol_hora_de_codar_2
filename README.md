Lógica de Programação II: Estruturas de Controle
Resolução de problemas focada em estruturas condicionais, comparações numéricas e processamento de listas de dados.

Desafios Resolvidos
Comparações e Máximos: Identificação do maior valor entre 2 e 3 números informados.

Análise de Sinais: Verificação de números positivos, negativos ou zero.

Lógica de Soma e Média: Cálculos envolvendo os maiores valores de uma lista e médias aritméticas.

Filtragem de Dados: Soma de valores baseada em condições (menores que 72) e validação de notas (entre 0 e 10).

Tomada de Decisão: Verificação de elegibilidade eleitoral e cálculo de peso ideal baseado em gênero e altura.

Micro Calculadora: Implementação de operações aritméticas básicas via menu de opções.

Conceitos Técnicos
Estruturas Condicionais: Uso intensivo de if, else if e when.

Operadores Lógicos: Aplicação de && (E) e || (OU) para validação de intervalos.

Listas e Iterações: Processamento de múltiplos valores informados pelo usuário.

Tecnologias
Linguagem: Kotlin

Paradigma: Imperativo / Estruturado
